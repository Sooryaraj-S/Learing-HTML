# 1-Week HTML Learning Roadmap

Day 1 – HTML Basics

What is HTML & how the web works

Structure of an HTML document

Tags, elements, attributes

Practice: Create a basic HTML page

Day 2 – Text & Formatting

Headings, paragraphs

Bold, italic, underline

Lists (ordered & unordered)

Practice: Simple content-based webpage

Day 3 – Links & Media

Anchor tags (internal & external links)

Images and attributes

File paths

Practice: Add navigation and images

Day 4 – Tables & Forms

Tables (rows, columns, headers)

Forms, input types, labels, buttons

Practice: Contact form layout

Day 5 – Semantic HTML

Semantic tags (header, footer, section, article, nav)

Importance of accessibility & clean structure

Practice: Semantic layout page

Day 6 – HTML Best Practices

Comments

Meta tags

SEO basics

Clean and readable HTML code

Day 7 – Mini Project & Revision

Build a simple personal webpage

Revise all concepts

Prepare notes for CSS learning.
